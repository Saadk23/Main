# User Guide - O.T.T.O ChatBot

**Step-by-step guide for using the chatbot - Aasan Urdu/English mein!**

---

## 🎯 Quick Start (Jaldi Shuru Karein)

### Step 1: Website Kholein

```
1. Browser kholein (Chrome, Firefox, Safari - koi bhi)
2. Address bar mein ye type karein: localhost:3000/index.html
   (Ya jo link admin ne diya ho)
3. Enter dabayein
```

### Step 2: Chat Icon Dhoondein

- **Screen ke neeche right corner** mein ek **purple color ka icon** dikhega
- Ye aise dikhta hai: 💬 (message icon)
- Is par **click karein**

### Step 3: Chat Window Khulega

Ek box khulega jis mein likha hoga:
```
O.T.T.O
Type your message...
```

### Step 4: Apna Sawal Likhein

- Neeche text box mein apna sawal type karein
- **Example:** "Swimming pool ki timings kya hain?"

### Step 5: Send Karein

Do tareeqe hain:
1. **Send button** (purple arrow) par click karein
2. Ya keyboard se **Enter** dabayein

### Step 6: Jawab Miley Ga

- "O.T.T.O is typing..." dikhega
- 1-2 second mein jawab aa jayega
- Jawab screen par dikhega

**Mubarak Ho! Aap ne pehla sawal pooch liya! 🎉**

## 📝 Tips for Best Results

### ✅ DO (Yeh Karein)

1. **Clear sawal poochein**
   - ❌ "Timings"
   - ✅ "Pool ki timings kya hain?"

2. **Simple language use karein**
   - Urdu ya English dono chalein ge
   - Mix bhi kar sakte ho

3. **Specific rahein**
   - ❌ "Facilities?"
   - ✅ "Gym ki facilities kya hain?"

4. **Complete sentences**
   - Helps the bot understand better

### ❌ DON'T (Yeh Na Karein)

1. **Bahut chhota sawal**
   - "Pool?" (too short)
   
2. **Ek saath kaafi sawaal**
   - Ek waqt mein ek sawal behtar hai

3. **Personal information**
   - Apna password ya CNIC number na likhen

4. **Ulti seedhi language**
   - Bot confusion mein aa sakta hai

## 🚫 Troubleshooting (Agar Mushkil Ho)

### Problem 1: Chat Icon Nahi Dikh Raha

**Solution:**
- Page refresh karein (F5 dabayein)
- Browser window bada karein
- Different browser try karein

### Problem 2: Message Send Nahi Ho Raha

**Solution:**
- Internet connection check karein
- Text box mein kuch likha hai ya nahi dekhen
- Page reload karein

### Problem 3: Jawab Nahi Aa Raha

**Check karein:**
1. Internet connected hai?
2. "O.T.T.O is typing..." dikh raha hai?
3. Wait karein 5-10 seconds
4. Agar phir bhi nahi aya, sawal dobara poochein

### Problem 4: Samajh Nahi Aa Raha

**Agar bot ka jawab samajh nahi aya:**
- Sawal dobara different words mein poochein
- Simple words use karein
- Ya admin office se direct poochein
