# 📱 Society Smart App: Troubleshooting Guide

A guide to resolving common issues with the official Resident Mobile Application.

## 1. Login Issues
* **Forgotten Password:** Use the "Forgot Password" link on the login screen to receive an OTP via your registered mobile number.
* **Account Locked:** If you enter the wrong PIN 5 times, your account will be locked for 30 minutes for security.

## 2. Common Fixes
| Issue | Solution |
| :--- | :--- |
| **Bill not showing** | Refresh the 'Payments' tab or clear App Cache. |
| **RFID Not Updating** | Ensure your maintenance dues are cleared. |
| **QR Code Error** | Update the App to the latest version from Play Store/App Store. |
