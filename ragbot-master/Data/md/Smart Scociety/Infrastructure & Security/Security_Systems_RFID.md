# 🛡 Advanced Security & RFID SOPs

## 1. Vehicle Access
* **RFID Tags:** Every resident's vehicle must display an RFID sticker for automated entry (Fee: Rs. 500).
* **Temporary Passes:** Guests must undergo biometric or phone verification at the main gate.

## 2. CCTV Surveillance
* The society is under 24/7 CCTV monitoring. To request video footage for incidents, a formal application must be submitted to the Admin Office.
